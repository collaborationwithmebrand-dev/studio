# **App Name**: VibrantBazaar

## Core Features:

- Dynamic Product Catalog: Display products with images, names, prices, and categories in a responsive grid layout.
- Product Search Functionality: Allow users to quickly find products by searching through their names.
- Festive Theme Engine: Switch between pre-defined festive themes that change the app's color scheme, navigation bar styling, and add dynamic visual effects.
- Admin Product Management: An administrator panel for adding, updating, and removing products from the store. This will require a persistent database.
- AI Product Description Tool: A generative AI tool in the admin panel to assist in creating compelling and relevant product descriptions.
- WhatsApp Order Placement: Facilitate direct orders from customers by generating and sending pre-filled messages via WhatsApp.
- Sales Tracking Dashboard: Display total orders and earnings statistics for administrators. This will also require a persistent database.

## Style Guidelines:

- Primary color: Energetic, deep blue (`#0D57F7`) to convey professionalism and vibrancy, aligning with the celebratory theme of the store.
- Background color: A subtle, cool light blue (`#E7EEFC`), providing a clean canvas that complements the primary color and maintains a bright, open feel for the festive edition.
- Accent color: A lively purple (`#624CE8`) analogous to the primary hue, offering a strong contrast for calls-to-action and important highlights, suitable for a festive visual appeal.
- Headline and body font: 'Poppins' (sans-serif) for its modern, precise, and clean aesthetic that is highly legible across various screen sizes, supporting both short product descriptions and UI elements.
- Utilize a consistent set of vibrant and expressive icons, ideally from a library like Font Awesome, to enhance product categories and administrative actions, particularly to complement the dynamic festive themes.
- Implement a responsive, intuitive grid-based product display that adapts seamlessly across different devices. A sticky navigation bar should ensure persistent access to search and administrative functions.
- Incorporate subtle hover effects on product cards and smooth transitions during theme changes. Light, celebratory animations for festive effects (like falling snow or festive lights) should be triggered by theme activation.
